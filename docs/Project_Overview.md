# Project Overview

## Objective

The objective of this project is to automate the complete Active Directory user lifecycle process using PowerShell.

The solution reduces manual administrative effort and standardizes user provisioning activities.

## Supported Operations

- User Creation
- User Modification
- User Deletion
- OU Assignment
- OU Movement
- EmployeeID Management
- EmployeeNumber Management
- Manager Assignment
- Group Membership Cloning
- Validation Handling
- Logging
- Scheduled Execution

## Benefits

- Reduced manual effort
- Faster provisioning
- Consistent user placement
- Reduced human error
- Centralized execution
- Improved auditability

## Technologies Used

- PowerShell
- Active Directory
- Windows Server
- ImportExcel Module
- Task Scheduler

## Architecture

HR Input File
↓
Validation
↓
PowerShell Processing
↓
Active Directory Operations
↓
Logging
